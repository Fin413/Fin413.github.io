Mandelbrot Explorer

This visalizer is a 2D representation of the Mandelbrot set, it displays all values within Mandelbrot's set that remain bounded within a given number of iterations (the detail).

The values within this set (zn+1 = z2n + c) form a infinite repeating fractal pattern known as an escape time fractal.

Click with your mouse to aim then change the zoom and resolution to explore the facinating patterns and formations!
